# Sentiment Analysis of Customer Reviews Using Deep Neural Networks

## Project Objective
This project builds an NLP model to classify IMDb movie reviews as positive or negative using deep neural networks.

## Dataset
IMDb Movie Reviews Dataset:
- 50,000 reviews
- 25,000 training samples
- 25,000 testing samples
- Binary labels: positive or negative

## Methods
The project uses:
- Text preprocessing
- Tokenization
- Padding
- Embedding layer
- Dense neural network
- Binary Cross-Entropy loss
- Adam optimizer

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## How to Run
pip install -r requirements.txt

python src/train.py

python src/evaluate.py
